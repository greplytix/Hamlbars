Ways to Contribute

  • Open an issue on github for any bugs/errors.
  • Open an issue on github for any missing features.
  • Fork our github repository, modify code, and submit a merge pull request.

To Test Contributions:

Clone your forked repository into one of the following directories, or place a symbolic link to the forked repository in one of the following directories:
  • Linux: ~/.config/sublime-text-3/Packages/User
  • OS X: ~/Library/Application Support/Sublime Text 3/Packages/User
  • Windows: %APPDATA%/Roaming/Sublime Text 3/Packages/User
