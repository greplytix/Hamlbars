Hamlbars is an open-source project aiming to provide syntax highlighting support for Sublime Text Editor. Hamlbars is a combination of haml and handlebars, and is useful for thorax, a leading framework for backbone.

Install Through Package Control
  1. Open the command palette. (ctrl+shift+p or command+shift+p)
  2. Type "install" and choose "Package Control: Install Package".
  3. Type "Hamlbars" and select the package by Greplytix.
  4. Ensure that "Hamlbars" is selected in the language menu.
